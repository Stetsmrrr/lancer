Системы контроля версий можно разделить на две группы: 1. Централизованные системы контроля версий (ClearCase) 2. Распределенные системы контроля версий (Git, Mercurial) Серверная операционная система – это специализированный тип ОС, который предназначен для управления аппаратным обеспечением сервера и предоставления различных сервисов другим компьютерам или устройствам в сети. В отличие от пользовательских операционных систем, серверные обычно оптимизированы для поддержки большого объёма параллельных процессов, высокой надёжности, стабильности и безопасности. Компания Microsoft предлагает ОС Windows Server – серверную операционную систему корпоративного класса с широкими возможностями управления хранением данных, приложениями и сетями. Компания Apple также имеет ОС для серверов macOS Server, содержащую возможности программирования сервера, а также управления и администрирования пользователей ОС macOS для персональных компьютеров. Кроме того, альтернативами являются ОС на базе Linux: Red Hat Enterprise Linux, Ubuntu Server и CentOS. Существуют также серверные ОС UNIX. Дистрибутив - программное обеспечение, которое содержит операционную систему и приложения, предназначенное для распространения и установки на компьютеры или другие устройства. Репозиторий - система, с помощью которой можно хранить, отслеживать и совместно разрабатывать код. Виртуализация — обособление части логических ресурсов вычислительной системы для выполнения изолированных процессов (или их совокупности) на одном и том физическом устройстве. Эта технология позволяет эффективно расходовать возрастающие возможности аппаратных платформ, распределяя их между несколькими программами или процессами, что широко используется в различных областях IT-индустрии. (создание изолированной программной среды (или нескольких таких сред) в рамках одного физического устройства)
